# 🚀 Day 06 — To-Do App (Basic)

Part of my 21 Days JavaScript Challenge.

## 📌 Overview
A simple to-do app where users can add and delete tasks dynamically.

## 🎯 Goal
Practice DOM manipulation and dynamic UI updates.

## 🧩 Features
- Add Task  
- Delete Task  
- Instant UI update  

## 🧠 Flow
1. Get input value  
2. Create list item  
3. Append to UI  
4. Remove task on delete  

## 🖥 Tech
HTML | CSS | JavaScript  

## 💡 Learning
DOM manipulation + dynamic elements creation
